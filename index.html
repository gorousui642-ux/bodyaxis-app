🎤 マイク許可の設定

  1. iPhone / Safari： 設定 → Safari → マイク → 許可
  2. iPhone / Chrome： 設定 → Chrome → マイク → 許可

※ 振動センサー（加速度計）はブラウザで自動的に使用されます。追加設定は不要です。

ブラキシズムとは

睡眠中の歯ぎしり・食いしばり。下顎振動子の高周波帯ズレは姿勢（低周波）と呼吸（中周波）の同期乱れと強い相関を示します。

STANDARD ONLY

無呼吸・呼吸状態の計測

睡眠中の呼吸パターンを解析し、無呼吸イベントを検出。中周波帯の位相データを取得します。

STANDARD ONLY

姿勢・呼吸・下顎 三層相関

三層同期指数を統合した総合スコア。Kuramoto秩序パラメータ r(t) をリアルタイム表示。

STANDARD ONLY

経時変化グラフ

週・月単位での位相同期スコア推移。介入効果の可視化と記録保存機能。
